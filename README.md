# Система за управление на библиотека

Проект по Обектно-ориентирано програмиране, реализиран на C++.

## Описание

Конзолно приложение за управление на библиотека, което позволява:

- управление на книги
- управление на потребители
- вход и изход от системата
- работа с файлове
- търсене и сортиране на книги
- администраторски команди

---

## Функционалности

### Работа с файлове
- `open <file>`
- `close`
- `save`
- `saveas <file>`

### Потребители
- `login`
- `logout`
- добавяне на потребители
- премахване на потребители

### Книги
- показване на всички книги
- подробна информация за книга
- търсене на книги
- сортиране на книги
- добавяне на книги
- премахване на книги

---

## Използвани технологии

- C++
- STL (`vector`, `map`)
- Работа с файлове (`fstream`)
- Exception handling
- Обектно-ориентирано програмиране

---

## Структура на проекта

- `Book` — информация за книга
- `User` — базов клас за потребители
- `Admin` — администратор
- `Client` — обикновен потребител
- `Library` — основна логика на системата
